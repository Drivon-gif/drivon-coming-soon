How to use with GitHub Pages:
1. Create a new public repository on GitHub (e.g. "drivon-coming-soon").
2. Upload both files (index.html and drivon-logo.png) into the root of that repository.
3. In the repository, go to Settings -> Pages.
4. Under Build and Deployment, choose "Deploy from a branch", then pick the "main" branch and "/" root folder.
5. Wait a minute, then open the URL GitHub shows you.
6. Point your drivoN.co.nz DNS records at the GitHub Pages URL as instructed in ChatGPT.
